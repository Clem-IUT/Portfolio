SAE 1.04 : Création de base de données

Contexte : Dans le cadre du BUT Informatique, j'ai réalisé un projet en binôme visant à réaliser un script SQL permettant la création et supprésion d'une base de données.

Objectifs du projet

  • Analyse du cahier des charges
  • Traduction des besoins en un diagramme de classes UML
  • Traduction du schéma en script SQL
  • Implémentation de la base de données via SQL Workbench

Compétence ciblée :

Concevoir et mettre en place une base de données à partir d'un cahier des charges client.

Outils utilisés

  • Langage PostgreSQL
  • Logiciel de gestion de base de données : SQL Workbench

Résultat

Une base données permettant d'organiser de façon efficace les données client les rendant plus faciles à manipuler.
