SAE 1.01 : Jeu du Snake

Contexte : Dans le cadre du BUT Informatique, j'ai réalisé un projet seul visant à reproduire le jeu du snake à travers différentes version allant de 1 à 4 que nous devions améliorer à chaque fois en respectant les nouvelles consignes données afin de remplir un cahier de tests pour chacune des versions pour finalement documenter la version finale.

Objectifs du projet

  • Analyse du cahier des charges
  • Implémentation des besoins en langage C
  • Rédaction du cahier de tests
  • Documentation de la version finale à l'aide de Doxygen

Compétences ciblées :

  • Développer des applications informatiques simples
  • Faire des essais et évaluer leurs résultats en regard des spécifications

Outils utilisés

  • Langage C
  • Langage Doxygen
  • Excel pour la rédaction des cahiers de tests

Résultat

Différentes versions d'une application en C répondant chacune à un cahier des charges propres ainsi qu'un cahier de tests pour chacune de ces versions.
